41. Padding controls the space inside an element between its content and border, while margin controls the space outside an element, creating distance from surrounding elements.
42. Set border-radius: 50%; on an image with a fixed width and height to make it circular.
43. Block elements take up the full width and start on a new line (e.g., "div", "p"), while inline elements only take up as much width as needed and stay in line with surrounding text (e.g., "span", "a").
44. em is relative to the font-size of the parent element, while rem is relative to the root "html" font size.
45. The z-index property controls the stack order of elements. Higher values place elements in front of lower values. It only works on positioned elements (relative, absolute, fixed).
46. The display property determines how an element is rendered, such as block, inline, flex, grid, or none (hiding the element).
47. absolute positions the element relative to its nearest positioned (non-static) ancestor, while relative positions it relative to its normal position.
48. CSS follows specificity rules, and the most specific rule applies. If rules have the same specificity, the later one in the stylesheet wins. The !important rule overrides all other rules.

50. :hover applies styles when the user moves the mouse over an element, while :active applies styles only when the element is clicked.
